    Data Download¶

    The haies pipeline requires several datasets from the IGN (Institut national de l'information géographique et forestière) to train models and compute indicators. The haies_download CLI tool simplifies fetching these datasets.

    Available Datasets¶

    The CLI provides commands to download imagery, elevation, and vector data.

    1. Aerial Imagery (BD Ortho)¶

    Download RGB or IRC imagery for a specific year and department. This is the primary input for the model.

    haies_download bdortho --output ./data/bdortho --year 2024 --department 17
    • --irc: Use this flag to download infrared imagery instead of RGB.
    • --in-parallel: Set to True (default) to download multi-part archives concurrently.

    2. LiDAR Elevation (Lidar HD)¶

    Download 1km x 1km LiDAR tiles (MNH - Modèle Numérique de Hauteur) that intersect a given Area of Interest (AOI).

    haies_download lidar-hd --geojson-file ./aoi.geojson --output ./data/lidar --compress
    • --compress: Compresses the downloaded .tif files into .jp2 format to save space (recommended).

    3. Hedge Database (BD Haie)¶

    Download the national hedge database and convert it to a GeoParquet file. This is used as the ground truth for training targets.

    haies_download bdhaie --output ./data/bdhaie.parquet

    4. Forest Mask (Foret V1)¶

    Download the national forest mask. This dataset is only used during the indicators computation step to ensure that only true hedges are analyzed, by filtering out large forested areas that might otherwise be misidentified.

    haies_download foretv1 --output ./data/foret.parquet

    5. Land Cover (OCS GE)¶

    Download the Occupation du Sol à Grande Échelle (OCS GE) dataset for building masks.

    haies_download ocs-ge --output ./data/ocsge.parquet --department 17 --year 2021

    Typical Workflow¶

    To prepare data for training in a new department:

    1. Download imagery: haies_download bdortho --output data/bdortho --department 17
    2. Download LiDAR: haies_download lidar-hd --geojson-file aoi.geojson --output data/lidar --compress
    3. Download Ground Truth: haies_download bdhaie --output data/bdhaie.parquet
    4. Download Forest Mask (optional): haies_download foretv1 --output data/foret.parquet (required only for Indicators Computation).

    Once downloaded, you can proceed to Label Preprocessing.
